# NLP-SENTIMENT-ANALYSIS-CUSTOMER-FEEDBACK
NLP-BASED SENTIMENT ANALYSIS PROJECT USING BERT EMBEDDINGS AND MACHINE LEARNING
## Project Overview
This project uses Natural Language Processing (NLP) and Machine Learning techniques to classify customer feedback into Positive, Neutral, and Negative sentiments.

## Technologies Used
- Python
- BERT Embeddings
- Logistic Regression
- Linear SVC
- Pandas
- NumPy
- Scikit-learn
- Power BI

## Key Features
- Data preprocessing and cleaning
- Feature extraction using BERT embeddings
- Sentiment classification
- Model evaluation and comparison
- Interactive dashboard for sentiment insights

## Results
The Linear SVC model achieved up to 97% classification accuracy and was selected as the final model.
